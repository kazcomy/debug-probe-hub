# Documentation Index

This directory is organized to avoid duplicated instructions.

## Ownership

- `docs/deploy.md`
  - Proxmox + cloud-init provisioning only.
  - VM creation and first-boot handoff only.
- `docs/operations.md`
  - Non-Proxmox/manual operation only.
  - Local install/start/update/troubleshooting.
- `docs/architecture.md`
  - Runtime model and locking/container behavior.
- `docs/api.md`
  - Endpoint contracts and request examples.
- `docs/configuration.md`
  - `config.yml` schema and extension workflow.

## Rule

- If a section belongs to another document, link to it instead of copying text.
